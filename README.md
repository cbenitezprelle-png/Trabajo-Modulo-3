# Trabajo Módulo 3

## Descripción
Proyecto correspondiente al módulo 3 que contiene scripts SQL y documentación técnica para la gestión y manipulación de bases de datos.

## Contenido del Proyecto

- **Trabajo m3.sql**: Script SQL con las consultas y procedimientos del módulo 3
- **trabajo modulo 3.docx**: Documentación detallada del trabajo realizado

## Requisitos Previos

- SQL Server o sistema de gestión de bases de datos compatible
- Cliente SQL para ejecutar los scripts
- Acceso de lectura/escritura a la base de datos

## Instalación

1. Clone el repositorio:
```bash
git clone https://github.com/cbenitezprelle-png/Trabajo-Modulo-3.git
```

2. Acceda a la carpeta del proyecto:
```bash
cd Trabajo-Modulo-3
```

3. Ejecute el script SQL en su cliente de base de datos
